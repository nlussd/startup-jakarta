Jakarta Start-Up Scene
Start-Up Location

Jakarta Start-Up Scene App created by ArcGIS Javascript API 4.6.

Visualize Start-Up Jakarta locations in aggregate and time-aware animation of Start-Up by year founded.

Data source: 2017 Start-Up Jakarta xlxs dataset compiled from Tech Crunchbase by Kiki Ahmadi. Some information might not be updated. Information in this app is not intended to be formal reference.
